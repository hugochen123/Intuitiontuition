== Description ==
BZ MultiSatilet is an multipurpose business WordPress themes which is work many business including stock exchange, stock market, business, finanace and many more.

== Frequently Asked Questions ==

= How to customize theme ? =

Just click on Appearance -> Customize
Now you can customize all section in theme

= How to Upgrade theme to pro ?=

Just click on Appearance -> Customize 
Now find upgrade to pro button, just click on it to upgrade pro version

= How to work on header phone and email? =

Just click on Appearance -> Customize -> bz-multisatilet Header Phone and Email

= How to work on header Social link? =

Just click on Appearance -> Customize -> bz-multisatilet Social Link

= How to work on slider? =

Just click on Appearance -> Customize -> bz-multisatilet home page slider
please use image dimesion 1600px*550px

= How to work on welcome section? =

Just click on Appearance -> Customize -> bz-multisatilet HomePage Welcome Section

= How to work on Footer Widgets ? =

Just click on Appearance -> Customize -> Widget

= How to work on Footer copyright ? =

Just click on Appearance -> Customize -> bz-multisatilet Footer
